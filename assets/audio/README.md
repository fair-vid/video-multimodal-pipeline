# 🎧 FAIR-VID Audio Guides

This directory contains audio explanations that accompany FAIR-VID notebooks.

These recordings provide a quick and intuitive introduction to each notebook, helping users understand what they are about to run before interacting with the code.

## 🎯 Purpose

Each audio file explains:
- what the notebook does
- what the user is expected to do
- how the data is handled
- how the step fits into the FAIR-VID pipeline

The goal is to improve accessibility, especially for:
- first-time users
- non-technical audiences
- live demonstrations
